# MyCar 🚗

Мобилно (PWA) приложение за управление на разходите по автомобила — по модела на Drivvo.
React 18 + TypeScript + Vite, Zustand (localStorage), CSS Modules, recharts. Тъмна/светла тема.

## Функционалности
- **История** — хронологична лента с всички записи, групирани по месеци; тап за редакция.
- **Записи**: Зареждане, Разход, Услуга/сервиз, Приход, Маршрут (A→B), Показание (километраж), Напомняне.
- **Двугоривни коли** — Резервоар 1/2 (напр. газ + бензин); разходът се смята отделно за всяко гориво.
- **Зареждане** с избор на гориво, цена/литър ⇄ обща стойност (авто), пълен резервоар, пропуснато зареждане, бензиностанция, шофьор.
- **Справки** с под-табове (Общ / Гориво / Разходи / Приход / Услуга) и филтър по период (този месец / тази година / всичко):
  - метрики „на ден" и „на км", баланс, среден разход, цена/км;
  - разбивка по резервоари (средно, нисък/висок/последен, общ обем, обща цена);
  - графики: месечни разходи (наслоена), сравнение на разходите, по бензиностанции, по категории, тренд на цената на горивото.
- **Напомняния** по дата/километраж с повторение; статуси просрочено/наближава/активно.
- **Гараж** — няколко автомобила с превключване.
- **PWA** — офлайн работа, инсталиране на телефон.

> Рекламите от Drivvo (QR скенер, dm и др.) умишлено не са пренесени.

## Стартиране
```bash
npm install
npm run dev            # http://localhost:5173
npm run dev -- --host  # тест от телефон в локалната мрежа
npm run build          # production (PWA / GitHub Pages)
```
Данните се пазят локално в браузъра (localStorage, ключ `mycar-store-v2`).
